# Qwen Code MCP Configuration

This directory contains MCP (Model Context Protocol) server configurations for Qwen Code.

## Available MCP Servers

### gcloud
Google Cloud Platform integration for managing GCP resources.

### run-long-command
Enables running long-running shell commands in the background without blocking the CLI.

## Setup

MCP servers are configured automatically when Qwen Code detects the `.qwen/mcp` directory.
