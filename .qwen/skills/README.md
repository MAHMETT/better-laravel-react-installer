# Qwen Code Skills

Skills are reusable capabilities that enhance Qwen Code's functionality for specific tasks or domains.

## Available Skills

No custom skills configured yet. Skills can be added by creating directories with `SKILL.md` files in this folder.

## Creating a Skill

1. Create a new directory in `.qwen/skills/`
2. Add a `SKILL.md` file with skill definition
3. Include any supporting files (scripts, templates, etc.)

## Example Skill Structure

```
.qwen/skills/
└── my-skill/
    ├── SKILL.md
    ├── scripts/
    └── templates/
```
