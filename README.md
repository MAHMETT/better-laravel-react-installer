# Better Laravel React Installer

Cross-platform shell installer script for the Better Laravel React starter kit.

## Quick Start

```bash
./install.sh
```

## Features

- **Interactive TUI**: Clean, modern terminal interface
- **Dependency Checking**: Automatic verification of PHP, Composer, and Node runtimes
- **Branch Selection**: Choose from predefined project branches
- **Runtime Selection**: Support for npm, pnpm, yarn, or bun
- **Fail-Safe**: Stops immediately on errors with clear messages
- **Cross-Platform**: Works on Linux and macOS

## Requirements

- PHP (8.1+)
- Composer
- Git
- Node.js with at least one package manager (npm/pnpm/yarn/bun)
