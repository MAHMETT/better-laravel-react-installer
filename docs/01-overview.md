# Overview

The Better Laravel React Installer is a modern CLI tool that provides an interactive TUI workflow for installing the Better Laravel React starter kit.

## Features

| Feature | Description |
|---------|-------------|
| **Interactive TUI** | Clean, modern terminal interface with proper colors and formatting |
| **CLI Commands** | Full command-line argument support for automation |
| **Dependency Check** | Automatic verification of PHP, Composer, Git, and Node runtimes |
| **Branch Selection** | Choose from predefined project branches |
| **Runtime Support** | npm, pnpm, yarn, or bun |
| **Fail-Safe** | Immediate stop on errors with clear messages |
| **Cross-Platform** | Compatible with Linux and macOS |

## Requirements

| Dependency | Version | Required |
|------------|---------|----------|
| PHP | 8.1+ | Yes |
| Composer | Latest | Yes |
| Git | Any | Yes |
| Node Runtime | npm/pnpm/yarn/bun | At least one |

## Installation Methods

1. **Global Install** — Copy to `/usr/local/bin/better-laravel`
2. **Local Use** — Run `./better-laravel` from project directory
3. **Remote Install** — Curl script directly
