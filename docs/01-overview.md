# Overview

The Better Laravel React Installer is a cross-platform shell script that provides a clean, interactive TUI workflow for installing the Better Laravel React starter kit.

## Features

- **Interactive TUI**: Simple, modern, and minimal terminal interface
- **Dependency Checking**: Automatic verification of required tools
- **Branch Selection**: Choose from predefined project branches
- **Runtime Selection**: Support for npm, pnpm, yarn, or bun
- **Safe Installation**: Fail-safe with immediate stop on errors
- **Cross-Platform**: Compatible with Linux and macOS

## Requirements

- PHP
- Composer
- Node.js runtime (npm, pnpm, yarn, or bun)
- Git
- Bash
